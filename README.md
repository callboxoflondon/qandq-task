
# QandQ Film Task


## Run on your computer

Clone project

```bash
  https://github.com/callboxoflondon/qandq-task.git
```

Go to project directory

```bash
  cd my-project
```

Install packages

```bash
  npm install
```

Run server

```bash
  npm run start
```

  
## Kullanılan Teknolojiler

**İstemci:** React, NextJS, TailwindCSS
